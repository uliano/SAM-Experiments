- [[Uart DMA]] — DMA-driven UART: 128-byte RX DMA ring, 64-byte TX chunks, DMAC channels 0+1
- [[Timebase]] — SysTick millis/micros, 1 ms resolution, PRIMASK-safe

- [[Startup SAMC21]] — vector table (31 peripheral IRQs), reset handler: .data copy / .bss zero / VTOR
- [[Syscalls]] — newlib `_write`/`_read` bridged to `Serial`
